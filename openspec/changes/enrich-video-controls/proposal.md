# Proposal: Enrich Video Controls

## Why

当前视频播放器底部仅有波形和进度条，缺少常见视频播放器的基础控制功能。用户需要更直观的时间显示、音量控制等功能来提升播放体验。

## What Changes

在视频播放器底部控制区域添加以下功能：

### 1. **时间显示**

- 显示当前播放时间 / 总时长（例如：00:35 / 10:24）
- 位置：进度条左侧或下方
- 样式：小字体，半透明白色

### 2. **音量控制**

- 音量图标（可点击静音/取消静音）
- 音量滑块（hover 或点击时显示）
- 位置：控制栏右侧
- 状态：显示当前音量级别（静音、低、中、高）

### 3. **播放速度控制**

- 速度选择按钮（0.5x, 0.75x, 1x, 1.25x, 1.5x, 2x）
- 位置：音量控制旁边
- 默认：1x
- 样式：文字按钮，当前速度高亮

### 4. **全屏按钮**

- 全屏/退出全屏切换
- 位置：控制栏最右侧
- 图标：fullscreen / fullscreen_exit

### 5. **布局优化**

- 将控制器分为两行：
  - **上层**：波形 + 进度条（保持现有）
  - **下层**：时间 | 播放/暂停 | 音量 | 速度 | 全屏
- 控制器背景：半透明黑色渐变
- 自动隐藏逻辑：保持现有的 3 秒自动隐藏

## Impact

### 修改文件

- `sentence_practice_screen.dart`：
  - 修改 `_buildVideoArea` 中的底部控制区域
  - 添加音量控制状态 `_volume`
  - 添加播放速度状态 `_playbackSpeed`
  - 添加全屏状态管理
  - 实现音量调节方法 `_setVolume`
  - 实现速度调节方法 `_setPlaybackSpeed`
  - 实现全屏切换方法 `_toggleFullscreen`

### 新增功能

- 音量控制（0.0 - 1.0）
- 播放速度控制（0.5x - 2x）
- 全屏模式支持
- 实时时间显示

### 用户体验提升

- 更专业的视频播放体验
- 更好的播放控制能力
- 符合用户对视频播放器的常规预期
