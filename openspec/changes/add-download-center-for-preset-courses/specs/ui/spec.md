## ADDED Requirements
### Requirement: 首页空课程引导
当本地不存在可学习课程时，应用必须 (MUST) 在首页展示空状态引导页面，并引导用户前往下载中心。

#### Scenario: 首页显示空状态引导
- **Given** 用户首次进入应用且本地课程列表为空
- **When** 首页完成数据加载
- **Then** 页面必须展示空状态视觉与引导文案
- **And** 页面必须展示“前往下载中心”主按钮

#### Scenario: 从空状态进入下载中心
- **Given** 用户位于首页空状态引导页
- **When** 用户点击“前往下载中心”
- **Then** 应用必须跳转到下载中心页面

### Requirement: 移除本地课程 Mock 默认源
应用必须 (MUST) 不再使用本地硬编码视频数据或 mock 课程作为首页和课程选择页的默认学习数据源。

#### Scenario: 无已安装课程时不显示 mock 课程
- **Given** 用户未导入且未下载任何课程
- **When** 用户打开首页或课程选择入口
- **Then** 系统不得显示内置 mock 课程卡片
- **And** 系统必须维持空状态并引导用户下载课程

### Requirement: 下载中心课程目录
应用必须 (MUST) 提供下载中心页面，用于展示可下载的预置课程列表并支持分类筛选。

#### Scenario: 查看可下载课程
- **Given** 用户进入下载中心
- **When** 课程目录加载完成
- **Then** 页面必须展示课程标题、标签、体积、封面、当前状态（未下载/下载中/已安装等）

#### Scenario: 分类筛选课程
- **Given** 下载中心展示多个分类标签
- **When** 用户点击某个分类
- **Then** 列表必须按该分类过滤显示课程

### Requirement: 课程下载与安装
应用必须 (MUST) 支持课程下载、校验与安装，并将安装结果接入本地学习流。

#### Scenario: 下载并安装成功
- **Given** 用户点击某课程下载
- **When** 文件下载完成且 hash 校验通过
- **Then** 应用必须完成本地安装并将课程标记为已安装
- **And** 该课程必须可在本地课程列表中被读取并学习

#### Scenario: 下载或安装失败
- **Given** 下载中断、hash 不匹配或解压失败
- **When** 任务进入失败状态
- **Then** 页面必须显示失败状态和重试入口
- **And** 系统不得保留不完整安装目录

### Requirement: 课程项左滑删除
应用必须 (MUST) 支持对下载中心中的单条课程执行左滑删除，并按状态执行正确清理。

#### Scenario: 删除下载中课程
- **Given** 某课程处于下载中或暂停中
- **When** 用户左滑并确认删除
- **Then** 系统必须取消任务并删除临时下载文件
- **And** 该课程状态必须回到未下载

#### Scenario: 删除已安装课程
- **Given** 某课程已安装
- **When** 用户左滑并确认删除
- **Then** 系统必须删除该课程安装目录与本地元数据
- **And** 课程不得继续出现在已安装可学习列表中

### Requirement: 下载状态恢复
应用必须 (MUST) 在应用重启后恢复下载中心的课程状态显示。

#### Scenario: 重启后恢复状态
- **Given** 用户在关闭应用前存在下载中/暂停/失败/已安装课程
- **When** 用户重新打开应用并进入下载中心
- **Then** 页面必须恢复这些课程的状态显示
