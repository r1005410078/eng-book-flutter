# 规格：视频播放器 UI 视觉样式

## MODIFIED Requirements

### Requirement: 播放按钮样式优化

播放按钮必须 (MUST) 呈现出更精致的视觉效果，以匹配现代 UI 设计风格。

#### Scenario: 播放按钮外观

- **Given** 视频处于暂停或交互状态显示控制层
- **Then** 播放按钮应为一个直径约 64dp 的圆形容器
- **And** 背景色应为白色带透明度 (Opacity ~0.2-0.3)
- **And** 边框应为极细的白色边框 (0.5dp 或 1dp，可选)
- **And** 图标应为纯白色，尺寸适中 (约 32-40dp)

### Requirement: 进度条样式优化

视频进度条必须 (MUST) 具有足够的视觉显著性和美观度。

#### Scenario: 进度条外观

- **Given** 视频控制层显示
- **Then** 进度条应位于视频区域底部边缘上方
- **And** 进度条的高度（厚度）应增加至约 4-6dp
- **And** 进度条应具有圆角边缘 (Radius)
- **And** 已播放部分的颜色应为主题橙色 (`0xFFFF9F29`)
