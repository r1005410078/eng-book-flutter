# 提案：视频播放器 UI 视觉微调

## 变更背景

用户提供了视频播放器设计参考图，指出需要优化播放按钮和进度条的视觉样式。目前的实现较为基础，不够精致。

## 目标

1.  **优化播放/暂停按钮**: 增大尺寸，调整透明度和图标大小，使其更接近参考图的“毛玻璃”或半透明叠加质感。
2.  **优化进度条**: 增加进度条高度（厚度），调整圆角，使其更明显且易于拖动（虽主要用于显示，但视觉上需要更强）。
3.  **顶部按钮**: 确保顶部设置和字幕按钮样式与整体风格一致（参考图中有两个明显的圆形按钮）。

## 范围

- `SentencePracticeScreen` 中的 `_buildVideoArea` 方法。

## 风险

- 无明显风险，纯 UI 调整。
