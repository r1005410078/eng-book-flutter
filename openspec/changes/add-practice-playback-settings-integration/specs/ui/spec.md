## ADDED Requirements
### Requirement: 播放设置与练习流程即时联动

系统必须 (MUST) 将播放设置接入练习播放流程，使设置变更在当前会话中即时生效。

#### Scenario: 语速设置即时生效
- **Given** 用户正在句子练习页面播放内容
- **When** 用户在播放设置中将语速从 1.0x 改为 0.75x
- **Then** 当前播放器应立即以 0.75x 播放
- **And** 返回练习页面后无需重启播放即可保持该语速

#### Scenario: 字幕显示与字号即时生效
- **Given** 用户正在查看字幕
- **When** 用户切换原文/译文开关或拖动字幕大小
- **Then** 练习页字幕显示状态与字号应立即更新

### Requirement: 播放设置持久化与恢复

系统必须 (MUST) 持久化播放设置，并在后续进入练习时恢复最近一次有效配置。

#### Scenario: 重进后恢复设置
- **Given** 用户已修改播放设置并退出练习
- **When** 用户再次进入任意练习会话
- **Then** 系统应恢复最近一次有效设置
- **And** 若持久化数据损坏或缺失，系统应回退默认值并继续可用

### Requirement: 播放行为冲突优先级

系统必须 (MUST) 定义并执行播放行为冲突优先级，确保单句循环与自动录音在同一状态机内可预测执行。

#### Scenario: 循环开启时优先完成循环
- **Given** 单句循环次数设置为 N
- **When** 当前句播放到句末
- **Then** 系统应先完成 N 次循环
- **And** 循环完成后再推进到下一句

#### Scenario: 自动录音在跟读模式下执行
- **Given** 自动录音开启且用户进入跟读流程
- **When** 系统进入录音阶段
- **Then** 应自动开始与结束录音
- **And** 不应影响主播放状态机推进

### Requirement: 录音能力降级处理

系统必须 (MUST) 在录音权限不足或平台不支持录音能力时安全降级，保持练习流程可继续。

#### Scenario: 权限缺失时降级
- **Given** 自动录音开启
- **And** 当前设备未授予录音权限
- **When** 系统进入录音阶段
- **Then** 系统应显示明确提示并跳过录音步骤
- **And** 跟读与播放流程应继续执行
