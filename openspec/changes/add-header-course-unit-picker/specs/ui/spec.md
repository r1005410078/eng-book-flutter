## ADDED Requirements
### Requirement: 练习页头部标题课程单元选择入口

系统必须 (MUST) 在短视频练习页头部提供可点击标题入口，用于打开课程与单元选择面板。

#### Scenario: 点击标题打开选择面板
- **Given** 用户正在短视频练习页
- **And** 页面已有当前课程与当前单元上下文
- **When** 用户点击头部标题区域
- **Then** 应显示课程与单元选择面板
- **And** 面板默认选中当前课程
- **And** 面板默认高亮当前单元

#### Scenario: 关闭面板不改变学习上下文
- **Given** 用户已打开课程与单元选择面板
- **When** 用户点击遮罩或关闭动作退出面板
- **Then** 当前课程、当前单元与当前句子必须保持不变

### Requirement: 面板内课程与单元切换

系统必须 (MUST) 支持在选择面板内切换课程并选择单元，选择完成后将练习页跳转到目标课程目标单元。

#### Scenario: 切换到其他课程并选择单元
- **Given** 用户已打开课程与单元选择面板
- **And** 面板中存在多个可学习课程
- **When** 用户选择课程 B 并点击课程 B 的某个单元
- **Then** 面板关闭并返回练习页
- **And** 练习页内容切换到课程 B 的被选单元
- **And** 头部标题更新为课程 B 与当前单元信息

#### Scenario: 切换成功后持久化学习进度
- **Given** 用户在选择面板完成课程或单元切换
- **When** 目标内容加载成功
- **Then** 系统必须保存最新学习进度（课程、单元、句子）
- **And** 后续重新进入学习流时应优先恢复到该最新进度
