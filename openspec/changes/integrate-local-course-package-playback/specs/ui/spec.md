## ADDED Requirements
### Requirement: 本地课程包课程列表
应用必须 (MUST) 支持从本地课程包目录读取可学习课程列表，而不是仅依赖 mock 课程。

#### Scenario: 展示本地课程
- **WHEN** 本地存在 `.runtime/tasks/*/package/course_manifest.json`
- **THEN** 课程选择页必须展示这些课程
- **AND** 每个课程项必须可进入课程详情页

### Requirement: 课程详情到学习流程打通
应用必须 (MUST) 打通课程详情页“开始学习”按钮到真实学习页面的导航流程。

#### Scenario: 开始学习
- **WHEN** 用户在课程详情页点击“开始学习”
- **THEN** 应用必须进入句子练习页
- **AND** 页面必须使用该课程包句子数据作为数据源

### Requirement: 阅读页与句子页共享课程上下文
应用必须 (MUST) 让阅读页与句子页共享同一课程上下文，避免一个页面使用本地数据、另一个页面使用 mock 数据。

#### Scenario: 句子页切到阅读页
- **WHEN** 用户在句子练习页点击阅读模式
- **THEN** 阅读页必须打开同一课程与当前句子位置
- **AND** 页面不得回退到无关 mock 数据

### Requirement: 本地媒体播放与时间轴同步
应用必须 (MUST) 根据课程包中的媒体类型播放本地资源，并按句子时间轴同步显示。

#### Scenario: 本地视频播放
- **WHEN** 课时媒体类型是 `video` 且媒体文件可用
- **THEN** 句子练习页必须使用本地视频文件初始化播放器
- **AND** 播放进度变化时必须更新当前句子高亮

#### Scenario: 本地音频播放
- **WHEN** 课时媒体类型是 `audio` 且媒体文件可用
- **THEN** 句子练习页必须使用本地音频文件初始化播放器
- **AND** 上一句/下一句与 seek 必须按句子时间轴生效

### Requirement: 课程包异常降级
应用必须 (MUST) 在本地课程包缺失或字段异常时提供可见降级提示并保持页面可用。

#### Scenario: 课程包文件缺失
- **WHEN** 课程包缺少 `course_manifest.json` 或 `lesson.json`
- **THEN** 应用必须展示错误/警告提示
- **AND** 应用不得崩溃
