## ADDED Requirements
### Requirement: 切换课程时必须持久化学习进度
应用必须 (MUST) 在用户完成课程/单元切换后持久化当前学习位置，用于下次进入时恢复。

#### Scenario: 上下切换后立即保存
- **Given** 用户正在首页小视频学习流中播放课程
- **When** 用户上下滑动切换到另一个课程/单元并切换完成
- **Then** 系统必须保存当前课程上下文与当前句子位置
- **And** 保存结果必须对应最后一次成功切换的目标课程

### Requirement: 重进应用必须恢复最近有效进度
应用必须 (MUST) 在重新进入学习流时恢复最近一次有效的学习进度。

#### Scenario: 重启后恢复到最近课程
- **Given** 用户已切换到课程 B 的某句并完成保存
- **When** 用户关闭并重新打开应用
- **Then** 首页必须恢复到课程 B 的已保存句子

#### Scenario: 已保存进度无效时降级
- **Given** 已保存进度指向的课程文件已不存在或句子 ID 无效
- **When** 用户重新打开应用
- **Then** 系统必须回退到首个可用课程的首句
- **And** 应用不得崩溃
