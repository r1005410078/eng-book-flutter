# 首页 UI 重构提案

## 目标

根据最新的设计稿，重构应用的首页（Home Screen）。目前的首页仅为临时占位符，新的设计将提供清晰的学习路径、进度概览和更具沉浸感的视觉体验。

## 背景

用户提供了高保真的首页 UI 设计稿，采用深色主题，强调学习进度和关卡式得学习路径。

## 变更范围

1.  **文件结构**:
    - 在 `lib/src/features/home/presentation/` 下创建独立的 `home_screen.dart` 和相关组件。
    - 移除 `lib/src/routing/app_router.dart` 中的临时 `HomeScreen` 类。
2.  **UI 变更**:
    - **全局主题**: 适配深色背景 (`#1a120b`) 和橙色强调色 (`#ff9f29`)。
    - **头部 (Header)**: 显示打卡天数、课程标题。
    - **进度卡片 (Header Card)**: 展示当前章节进度、今日目标完成度和总练习次数。
    - **学习路径 (Learning Path)**: 垂直滚动的关卡列表，包含已完成、正在进行（高亮）和未解锁状态。
    - **动作按钮**: 底部悬浮的“阶段测试”入口。

## 影响

- 将作为应用的新启动页。
- 路由配置需要更新以指向新的 `HomeScreen` 文件。
