# 首页 UI 规范

## 1. 颜色系统

- **背景色**: `#1a120b` (深棕/黑)
- **卡片背景**: `#2c241b` (稍浅的深棕)
- **主色/强调色**: `#ff9f29` (亮橙色，用于按钮、进度条、高亮状态)
- **文本色**:
  - 主要: `#ffffff` (纯白)
  - 次要: `Colors.white.withOpacity(0.6)`
  - 暗淡: `Colors.white.withOpacity(0.3)`

## 2. 布局结构

使用 `CustomScrollView` 或 `ListView` 实现整体滚动，背景固定。

### 2.1 头部区域 (Header)

- **左侧**: "12 天" (火苗图标 + 数字)，字体粗体，白色。
- **中间**: "商务英语基础"，白色，16sp，居中。
- **右侧**: 书本图标 (IconButton)，白色。

### 2.2 进度统计卡片 (Progress Card)

- **容器**: 圆角矩形 (Radius 24)，背景色 `#2c241b`。
- **左侧**: 圆环进度条 (30%)，橙色进度，灰色底。中间显示 "30%"。
- **右侧内容**:
  - "当前章节": 小字，灰色。展示 "职场会议表达"。
  - "今日会话 12/100": 右上角标签。
  - "450 次练习": 白色数字 + 文字，下方有一条水平进度条。

### 2.3 学习路径 (Learning Path)

垂直排列的节点，每个节点代表一个单元。

- **连接线**: 节点之间有虚线连接 (`CustomPaint` 或 `DashedLineWidget`)。
- **节点类型**:
  1.  **已完成 (Completed)**:
      - 圆形头像（如佩奇），外圈带金色/橙色光晕。
      - 右上角有圆形对钩图标。
      - 下方显示标题 "基础问候"。
  2.  **进行中 (Active)**:
      - 视觉焦点，尺寸较大。
      - 中心: 黑色播放三角形，背景黑色圆圈，外层有层层扩散的橙色光晕。
      - 上方标签: "正在学习" (橙色胶囊背景，黑色文字)。
      - 下方标题: "职场会议表达" (大号白色粗体)。
      - 副标题: "单元 2 / 15" (灰色小字)。
  3.  **未开始 (Locked/Pending)**:
      - 圆形虚线边框，内部为灰色/暗淡的人物头像。
      - 下方标签: "未开始" (灰色胶囊)。
      - 下方标题: "邮件往来技巧"。
  4.  **特殊/锁定节点**:
      - 普通图标（书本）或锁图标。
      - 五角星里程碑节点。

### 2.4 底部悬浮/列表尾部

- **阶段测试**: 位于列表最下方的一个独立卡片/按钮。
  - 样式: 深色卡片背景，带圆角，中间有奖杯图标和文字 "阶段测试"。

## 3. 交互

- **点击进行中节点**: 导航至 `SentencePracticeScreen` (传递对应 ID)。
- **点击头部右侧图标**: 可能是目录或设置，暂时预留。
