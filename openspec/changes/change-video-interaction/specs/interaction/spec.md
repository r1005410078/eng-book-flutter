# 规格：视频区域点击交互

## MODIFIED Requirements

### Requirement: 点击切换播放状态

点击视频显示区域（非特定功能按钮区）必须 (MUST) 触发播放或暂停操作，而不是仅切换控制层的可见性。

#### Scenario: 视频播放中点击

- **Given** 视频正在播放
- **And** 控制层可能处于显示或隐藏状态
- **When** 用户点击视频空白区域
- **Then** 视频应暂停
- **And** 控制层应显示

#### Scenario: 视频暂停时点击

- **Given** 视频已暂停
- **When** 用户点击视频空白区域
- **Then** 视频应恢复播放
- **And** 控制层应显示（随后根据播放状态自动隐藏）
