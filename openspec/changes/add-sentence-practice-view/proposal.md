# 提案：添加句子练习视图

## 摘要

实现一个专注的“句子练习”视图（如设计图所示），以支持对单句进行强化 100LS 训练。该视图将提供丰富的上下文，包括视频背景、音标、翻译和语法注释。

## 动机

100LS 方法要求对特定句子进行反复听和说。当前的 `AudioPlayer` 可能主要设计用于连续播放。用户需要一种专用模式来一次专注于一个句子，查看语法和翻译等详细信息，并录制他们的声音进行对比。

## 建议方案

- 创建一个新屏幕 `SentencePracticeScreen`。
- 实现带有视频背景的沉浸式 UI。
- 添加对显示解析信息（音标、翻译、语法注释）的支持。
- 集成专门用于单句循环的录音和播放控制。
