# 规格：视频播放控制优化

## MODIFIED Requirements

### Requirement: 播放控制自动隐藏

在视频播放期间，所有覆盖在视频上的控制元素必须 (MUST) 根据用户交互自动显示或隐藏。

#### Scenario: 播放时自动隐藏

- **Given** 视频正在播放中
- **And** 控制层当前处于显示状态
- **When** 用户在 3 秒内没有任何触屏操作
- **Then** 控制层（包括顶部栏、中间播放按钮、底部进度条、底部控制栏）应平滑淡出（Fade Out）至完全隐藏

#### Scenario: 点击屏幕切换显示状态

- **Given** 视频正在播放中
- **When** 用户点击视频区域的任意空白处
- **Then** 如果控制层当前隐藏，则立即显示（Fade In）
- **And** 如果控制层当前显示，则立即重置 3 秒计时器

### Requirement: 视频进度条

在视频区域或屏幕底部必须 (MUST) 添加进度条，用于显示视频播放进度。

#### Scenario: 显示进度条

- **Given** 控制层处于显示状态
- **Then** 屏幕底部（或视频底部）应显示一个线性进度条
- **And** 进度条应实时更新当前播放位置

#### Scenario: 隐藏进度条

- **Given** 控制层处于隐藏状态
- **Then** 进度条应不可见
